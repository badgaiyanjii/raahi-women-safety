# 🚨 Raahi – Women Safety Device

Raahi is a hardware + software based women safety system using GPS & GSM technology.

## 🔥 Features
- One-click SOS trigger
- Real-time location sharing
- SMS + Call alerts
- Loud buzzer alert
- Mobile app integration

## ⚙️ Tech Stack
- Arduino Nano
- SIM800L GSM
- Neo-6M GPS
- Android (Kotlin)
- Firebase (optional)

## 🔌 How it Works
1. User presses SOS button
2. GPS fetches location
3. GSM sends SMS & calls
4. Buzzer alerts nearby people

## 📱 App Features
- Emergency alerts
- Fake shutdown mode
- Chatbot (planned)

## 🚀 Future Scope
- AI-based danger detection
- Wearable integration
- Cloud tracking dashboard
